Microservices - Web Server

Overview

  • Defines a simple web-application for accessing insurance industry code data.
  • All the code information is fetched via a RESTful interface to a Lookup microservice.

The Demo

  • Eureka Dashboard: http://localhost:1111
  • Check applications registered: http://localhost:1111/eureka/apps/
  • Fetch code #1741: /codes/cacw/1741
  • Code Search
